基金监控 Pro

📊 实时持仓监控 - 显示基金前15大持仓股票的实时涨跌幅

🌍 多市场支持 - 支持A股、港股、美股
📱 手机端适配 - 响应式设计，手机访问体验良好
🔄 自动刷新 - 可设置刷新频率（5-60秒）

ETF联接基金说明
ETF联接基金会自动识别并从跟踪指数计算涨跌幅，无需手动配置。
如果基金名称中包含支持的指数名称（如"沪深300"、"中证白酒"等），系统会自动匹配对应的指数代码。

本工具提供的基金估算涨跌幅仅供参考，不构成投资建议
估算值基于季报持仓数据，可能存在滞后
实际净值以基金公司披露为准
